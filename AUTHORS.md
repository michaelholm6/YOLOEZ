# Authors and Contributors

This file lists the authors and contributors to **YOLOEZ**.

YOLOEZ is an open-source, GUI-based application for labeling data, training models, and running inference with Ultralytics YOLO11 models.  
The project was developed at **Purdue University** and is published as open-source research software.

## How to Be Added

If you have made a substantive contribution (e.g., code, tests, documentation, or significant design input):

1. Add your name to the **Contributors** section above
2. Include your ORCID iD if you have one (recommended)
3. Submit the change as part of your pull request or in a follow-up PR

Minor contributions (such as typo fixes) may not require inclusion, but contributors are welcome to add themselves if they wish.
